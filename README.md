This is a map of the world of the Outer Reincarnation CYAO, made by reddit user 3_tankista.

The CYOA's latest reddit thread (as of this writing): https://www.reddit.com/r/makeyourchoice/comments/1mwbkpv/outer_reincarnation_cyoa_update_1/
The CYOA on imgchest: https://imgchest.com/p/9p4n8wvvm4n

This map uses material made availible, by tankista, in his Google Drive folder (see his reddit thread for the link).
It displays the historical world maps in a google maps like style, with an added timeline slider, to better see the shifting borders over the millenium of outsider interference.

This project was made with ReactJS and besides native React only uses the external library https://www.npmjs.com/package/react-map-interaction.
